Seguridad

Tu seguridad y privacidad son nuestra prioridad

Este documento explica de forma sencilla cómo protegemos tu información cuando usas nuestro bot de facturación.

¿Qué datos guardamos?

Guardamos únicamente lo necesario para que puedas emitir facturas electrónicas:

Datos de tu negocio

RUC, razón social, dirección, establecimiento, etc.

Productos y clientes

Lo que tú registres para tus facturas.

Tu firma electrónica

El archivo .p12 que nos envías para firmar las facturas ante el SRI.

Contraseña de tu firma

La que usas para abrir ese archivo.

Todo esto se usa exclusivamente para brindarte el servicio de facturación. Nada más.

¿Usamos tus datos para algo más?

No.

Tus datos no se venden, no se comparten con terceros ni se usan para publicidad. Solo los utilizamos para:

1

Emitir tus facturas electrónicas

Cada factura que emites pasa por nuestro sistema de forma segura.

2

Mantener tu inventario y clientes organizados

Para que puedas facturar más rápido y sin errores.

3

Darte soporte cuando lo necesites

Para resolver tus dudas y ayudarte con el servicio.

Video de seguridad

Espacio para agregar un video explicativo

¿Quién puede ver mi contraseña?

Nadie.

Tu contraseña de la firma electrónica está protegida así:

1

Se guarda encriptada

Usamos un sistema de encriptación de nivel bancario (AES-256). Aunque alguien accediera a la base de datos, vería solo texto cifrado, no tu contraseña real.

2

Solo se usa cuando tú facturas

La contraseña se desencripta únicamente en el momento de firmar una factura, y solo para tu propia factura. No queda guardada en texto legible en ningún lado.

3

No la vemos nosotros

Ni los desarrolladores ni el personal del servicio pueden ver tu contraseña. El sistema la procesa de forma automática y segura.

¿Quién puede usar mi firma electrónica?

Solo tú.

Tu archivo de firma (.p12) se usa únicamente para firmar las facturas que tú emites a través del bot. No se puede usar para:

  • Firmar facturas de otros usuarios.
  • Firmar documentos que no sean tus facturas.
  • Cualquier otro propósito.

Además, la firma se valida contra tu RUC. Si el certificado no coincide con tu empresa, el sistema no la acepta.

Tu firma, tu control

Cada factura requiere tu autorización explícita a través de WhatsApp.
